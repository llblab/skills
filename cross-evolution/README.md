# Cross-Evolution Skill

Horizontal Gene Transfer protocol for autonomous agent skills with gene-meme equivalence, deep transportable genes, readable decomposed audits, soft proactivity, and JSON-first observation state.

## Platform Support

- 'Linux'
- 'MacOS'

## Axioms

- 'Gene == Meme': A gene is accepted only if it behaves like a transmissible meaning-bearing meme.
- 'Atomic Independence': Skills must not hard-code sibling skills, concrete projects, private repositories, or cross-skill glue.
- 'Agent as Orchestrator': Synergies are managed by the AI agent.
- 'Generic Synergy': Skills compose through neutral contracts and lenses, not stack mirrors.
- 'Pragmatic Evolution': Occam's Razor — no evolution for the sake of evolution.
- 'Soft Proactivity': Detect freely, suggest often, mutate narrowly, never farm scores.
- 'Memetic Recombination': Gene-memes can be decomposed, recombined, and composed into emergent new gene-memes.
- 'Horizontal Gene Transfer Mode': Soft proactivity transfers at most one donor gene-meme into one recipient before stopping.

## Observation Scripts

```bash
scripts/audit-cross-evolution.sh --root ~/.agents/skills
scripts/inspect-skill.sh brainstorming --root ~/.agents/skills
scripts/inspect-gene.sh atomic-independence --root ~/.agents/skills
```
