# Cross-Evolution Skill

Horizontal Gene Transfer protocol for autonomous agent skills with deep transportable genes, readable decomposed audits, and a JSON-first observation architecture direction.

## Platform Support

- 'Linux'
- 'MacOS'

## Axioms

- 'Atomic Independence': Skills must not hard-code sibling skills, concrete projects, private repositories, or cross-skill glue.
- 'Agent as Orchestrator': Synergies are managed by the AI agent.
- 'Generic Synergy': Skills compose through neutral contracts and lenses, not stack mirrors.
- 'Pragmatic Evolution': Occam's Razor — no evolution for the sake of evolution.

## Observation Scripts

```bash
scripts/audit-cross-evolution.sh --root ~/.agents/skills
scripts/inspect-skill.sh brainstorming --root ~/.agents/skills
scripts/inspect-gene.sh atomic-independence --root ~/.agents/skills
```
